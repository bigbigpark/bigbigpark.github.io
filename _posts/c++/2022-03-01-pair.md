좌표 묶을 때 좋은 C++ STL 소개 <br/>

## 라이브러리 인클루드
~~~c++
#include <pair>
#include <vector>
#include <queue>

// 큐나 벡터의 push_back()으로 많이 응용함!!
~~~

<br/>

## Example

~~~c++
pair<int, int> p;
p = make_pair(3, 3);
p = {3, 3};
~~~

<br/>

조금 더 응용을 하면? <br/>

~~~c++

vector< pair<int,int> > v;
v.push_back(make_pair(2,2));

queue< pair<int,int> > q;
q.push(make_pair(2,2));

~~~
