## MFC란?

MFC는 Microsost Foundation Class Library이다. 말 그대로 Microsoft에서 만든 C++ 기반의 Class 라이브러리다. 그렇다면 이거를 왜 만들었을까?

하지만 얘네들을 어떻게 만들 수 있는 지 고민해 본 적이 있었던가. MFC를 설명하기 앞서 이렇게 생긴 프로그램은 통상 Windows 응용프로그램이라고 불린다. <br/>
또한 컴퓨터 리소스를 확인하는 작업관리자 역시 Windows 응용프로그램 중 일부다. <br/>
이 프로그램은 C언어를 기반으로 하고 있고, OS에서 제공하는 API 역시 C언어다. 여기까지 설명을 보아하니 MFC는 이런 것들을 만드는 것과 연관성이 있어 보인다.

## MFC의 등장

C++ 언어를 사용하는 경우 API 함수들이 모두 C언어 기반이기 때문에 C++ 형식으로 재구성해야 하는데, 개수가 천문학적으로 많아서 이는 거의 불가능하다고 보면 된다. 따라서 Microsoft에서 C++ 개발자들을 위해 API 함수와 API 함수가 사용하는 구조체들을 모아서 Class를 미리 만들어 두었다. (마소 짱짱맨....) 

<br/>

이 클래스 집합을 **MFC**라고 부른다. 하지만 API가 너무 많고, 시간이 지날수록 추가되는 API가 있기 때문에 MFC가 모든 API를 담고 있진 않는다.
