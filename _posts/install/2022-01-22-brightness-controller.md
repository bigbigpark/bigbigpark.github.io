<br/>


화면 밝기 조절하는 거를 설치해보자!

<br/>

## 1. 저장소 추가 (ADD PPA)

~~~bash
$ sudo add-apt-repository ppa:apandada1/brightness-controller
~~~

<br/>

## 2. 패키지 업데이트 및 설치

~~~bash
$ sudo apt update
$ sudo apt install brightness-controller
~~~

<br/>

## 3. 실행

~~~bash
$ brightness-controller
~~~

<br/>

## 4. 삭제

~~~bash
$ sudo apt purge brightness-controller
~~~
