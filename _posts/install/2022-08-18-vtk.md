VTK 7.1.0 버전을 설치해보자!<br/>

## How to install VTK

~~~bash
$ wget http://www.vtk.org/files/release/7.1/VTK-7.1.0.tar.gz
$ tar -xf VTK-7.1.0.tar.gz
$ cd VTK-7.1.0 && mkdir build && cd build
$ cmake ..
$ make                                                                   
$ sudo make install
~~~
