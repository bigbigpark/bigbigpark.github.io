VTK 7.1.0 버전을 설치해보자!<br/>

## How to install VTK

Download source file from [here](https://vtk.org/download/)

~~~bash
$ tar -xf VTK-8.2.0.tar.gz
$ cd VTK-8.2.0 && mkdir build && cd build
$ cmake ..
$ make                                                                   
$ sudo make install
$ sudo ldconfig
~~~
