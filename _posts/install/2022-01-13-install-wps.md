<br/>

## Installation

사이트에 들어가서 deb 파일을 받아준다

~~~
https://www.wps.com/download/
~~~

<br/>

설치 경로에 가서 아래와 같은 명령어로 설치

~~~bash
$ sudo dpkg -i wps-office_11.1.0.10702.XA_amd64.deb
~~~

끝!
